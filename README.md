# Análisis de Siniestros de Tránsito en Uruguay

Este proyecto realiza un análisis exploratorio de datos sobre siniestros de tránsito ocurridos en Uruguay. El objetivo es identificar patrones y características relevantes, como el día de la semana con mayor cantidad de accidentes y la gravedad de los resultados.

## 📊 Descripción del Proyecto

Utilizando un dataset de siniestros, se exploran variables como:

- Día de la semana del accidente
- Tipo de resultado (heridos leves, graves, fallecidos)
- Relación entre el día y la gravedad de los siniestros

## 🧰 Herramientas Utilizadas

- **R**: lenguaje principal para el análisis
- **RStudio**: entorno de desarrollo
- **ggplot2** y **dplyr**: librerías para visualización y manipulación de datos
- **stringr**: limpieza y normalización de cadenas de texto

## 📈 Ejemplos de Análisis Realizados

- Gráfico de barras apiladas mostrando la cantidad de accidentes por día y tipo de resultado
- Limpieza y ordenamiento de los días de la semana
- Filtro por resultados más relevantes (heridos o fallecidos)
